# guizhan-resources-ci

Guizhan Resources CI runner scaffold for local integration and workflow contract testing.

## Overview

This repository provides the runner infrastructure and CI contract definitions for the Guizhan Resources auto-build system. It enables automated plugin building, artifact manifest generation, and status callbacks.

## Development

Install dependencies:

```bash
pnpm install
```

Run verification:

```bash
pnpm lint
pnpm type-check
pnpm test
pnpm actionlint
```

## Contracts and Examples

For detailed contract specifications, including failure mapping, see `docs/contracts.md`.

Example payloads for integration testing:
- `examples/build-payload.json`: The trigger payload sent to the CI workflow.
- `examples/callback-payload.json`: The status callback payload sent back to the API.
- `examples/runner-manifest.json`: The manifest generated by the runner upon build completion.

## MVP Exclusions

- Private repository support.
- OAuth / GitHub App authentication.
